# 💰 Smart Budget & Expense Optimization System

A full-stack web application that helps users track, analyze, and optimize their daily expenses using data-driven insights and rule-based recommendations.

---

## 🚀 Features

- Track income and expenses  
- Categorize spending (Food, Travel, Shopping, etc.)  
- Set monthly budgets  
- Visualize data using charts  
- Get smart spending recommendations  
- Forecast future expenses  

---

## 🧠 Optimization Approach

This system uses rule-based and heuristic techniques:

- 50/30/20 budgeting rule  
- Threshold-based alerts for overspending  
- Budget adjustment suggestions  
- Financial health scoring  
- Moving average forecasting  

---

## 🏗️ Tech Stack

- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: SQLite  

---

## 📌 Description

The system analyzes user spending patterns and provides actionable insights to improve financial management and savings.
